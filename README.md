# Installation

```
cd
sudo apt update
sudo apt install -y git
git clone https://github.com/zuzu59/NodeMCU_cours.git
cd NodeMCU_cours
./install.sh
```

# Utilisation
Pour démarrer l'IDE pour le NodeMCU, simplement exécuter le script:

```
./start.sh
```
